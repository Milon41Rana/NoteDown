# **App Name**: NoteDown

## Core Features:

- Note Creation: Allow users to create and write notes within the application using a rich text editor.
- Note Organization: Enable users to organize notes into folders or categories.
- PDF Generation: Convert selected notes into a PDF document with customizable formatting.
- PDF Download: Allow users to download the generated PDF to their local device.

## Style Guidelines:

- Primary color: A serene blue (#64B5F6) to promote focus and clarity.
- Background color: A very light, desaturated blue (#F0F8FF) to provide a comfortable writing environment.
- Accent color: A warm orange (#FF7043) to highlight key actions like PDF conversion and download.
- Font pairing: 'Inter' (sans-serif) for both headings and body text, to maintain a modern and readable interface.
- Use a clean, minimalist layout to prioritize content and reduce distractions.
- Employ simple, intuitive icons to represent actions and navigation elements.
- Incorporate subtle animations for smooth transitions and feedback during PDF generation and downloading.
